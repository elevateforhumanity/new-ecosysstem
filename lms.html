Integration & Embed Codes

🔗 LMS Embed Code

Copy this code to embed the LMS on your website:

<iframe src="https://lms.elevateforhumanity.org/embed" width="100%" height="600" frameborder="0" allowfullscreen> </iframe>

🔌 API Endpoint

REST API endpoint for integration:

https://api.elevateforhumanity.org/v1/lms

📜 JavaScript SDK

Include our JavaScript SDK:

<script src="https://cdn.elevateforhumanity.org/lms-sdk.js"></script> <script> EFH_LMS.init({ apiKey: 'your-api-key', environment: 'production' }); </script>

API Documentation

🔐 Authentication

API authentication header:

Authorization: Bearer your-api-key-here

📡 Sample API Call

Get student progress:

curl -X GET \ 'https://api.elevateforhumanity.org/v1/students/123/progress' \ -H 'Authorization: Bearer your-api-key' \ -H 'Content-Type: application/json'

🔔 Webhook URL

Configure webhooks for real-time updates:

https://your-domain.com/webhooks/efh-lms
