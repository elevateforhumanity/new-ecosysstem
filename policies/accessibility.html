Elevate for Humanity is committed to ensuring digital accessibility for people with disabilities. We continually improve the user experience for everyone and apply relevant accessibility standards.

We aim to conform to the Web Content Accessibility Guidelines (WCAG) 2.2 Level AA standards published by the World Wide Web Consortium (W3C).

Accessibility Features

Navigation & Structure

  • Skip Links: Skip to main content and navigation
  • Keyboard Navigation: Full site navigation using only keyboard
  • Focus Indicators: Visible focus states for all interactive elements
  • Logical Tab Order: Sequential navigation through page elements
  • Heading Structure: Proper heading hierarchy (H1-H6)

Visual Design

  • Color Contrast: Minimum 4.5:1 contrast ratio for normal text
  • Enhanced Contrast: 7:1 contrast ratio for important elements
  • Color Independence: Information not conveyed by color alone
  • Scalable Text: Text can be enlarged up to 200% without loss of functionality
  • Responsive Design: Adapts to different screen sizes and orientations

Content & Media

  • Alternative Text: Descriptive alt text for all images
  • Video Captions: Closed captions for all video content
  • Audio Descriptions: Available for visual content in videos
  • Transcripts: Text transcripts for audio and video content
  • Clear Language: Plain language and clear instructions

Forms & Interaction

  • Form Labels: Clear labels associated with all form controls
  • Error Identification: Clear error messages and correction guidance
  • Input Assistance: Help text and format requirements
  • Time Limits: Adjustable or no time limits on forms
  • Error Prevention: Confirmation for important actions

Assistive Technology Compatibility

Our website is designed to work with assistive technologies including:

  • Screen Readers: JAWS, NVDA, VoiceOver, TalkBack
  • Voice Recognition: Dragon NaturallySpeaking
  • Keyboard Navigation: Switch devices, alternative keyboards
  • Magnification: ZoomText, built-in browser zoom
  • Mobile Accessibility: iOS and Android accessibility features

How to Request Accommodations

  1. Contact Us Early: Request accommodations at least 48 hours in advance when possible
  2. Describe Your Needs: Let us know what specific accommodations would be helpful
  3. Provide Documentation: If required, provide documentation of your disability
  4. Work Together: We'll work with you to determine the most effective accommodations
  5. Follow Up: Let us know if accommodations are working or need adjustment

Accessibility Testing

We regularly test our website accessibility using:

  • Automated Testing: WAVE, axe, Lighthouse accessibility audits
  • Manual Testing: Keyboard navigation, screen reader testing
  • User Testing: Feedback from users with disabilities
  • Expert Review: Accessibility specialist evaluations

We are continuously working to improve accessibility. Current known issues and planned improvements include:

  • Ongoing enhancement of form validation messages
  • Improving video player accessibility controls
  • Adding more descriptive link text throughout the site
  • Enhancing mobile accessibility features

Some content on our website may be provided by third-party vendors. We work with our vendors to ensure their content meets accessibility standards, but if you encounter issues with third-party content, please let us know so we can address it with the vendor.

Standards & Guidelines

Our accessibility efforts are guided by:

  • WCAG 2.2 Level AA: Web Content Accessibility Guidelines 2.2
  • Section 508: U.S. federal accessibility standards
  • ADA: Americans with Disabilities Act requirements
  • WIOA Section 188: Nondiscrimination and equal opportunity provisions
