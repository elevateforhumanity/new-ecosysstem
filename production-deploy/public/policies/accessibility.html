Accessibility Statement

Our Commitment

Elevate for Humanity is committed to ensuring digital accessibility for people with disabilities. We are continually improving the user experience for everyone and applying the relevant accessibility standards to achieve these goals.

Conformance Status

We aim to conform to the Web Content Accessibility Guidelines (WCAG) 2.2 Level AA standards. These guidelines explain how to make web content more accessible for people with disabilities and user-friendly for everyone.

Our website has been designed and tested to meet these standards, including:

  • Perceivable: Information and user interface components are presentable in ways users can perceive
  • Operable: User interface components and navigation are operable
  • Understandable: Information and operation of user interface are understandable
  • Robust: Content can be interpreted reliably by a wide variety of user agents, including assistive technologies

Accessibility Features

Our website includes the following accessibility features:

  • Keyboard Navigation: All interactive elements can be accessed using keyboard navigation
  • Screen Reader Support: Proper heading structure, alt text for images, and semantic markup
  • Color Contrast: Text and background colors meet WCAG AA contrast requirements
  • Responsive Design: Content adapts to different screen sizes and zoom levels up to 200%
  • Focus Indicators: Visible focus indicators for keyboard navigation
  • Form Labels: All form inputs have associated labels
  • Skip Links: Skip navigation links for screen reader users
  • Descriptive Links: Link text describes the destination or purpose

Assistive Technologies

Our website is designed to be compatible with assistive technologies, including:

  • Screen readers (JAWS, NVDA, VoiceOver)
  • Voice recognition software
  • Keyboard-only navigation
  • Screen magnification software
  • Switch navigation devices

Known Limitations

Despite our efforts to ensure accessibility, there may be some limitations. We are actively working to address any issues and improve accessibility across our digital platforms.

Alternative Formats

If you need information from our website in an alternative format, we can provide:

  • Large print documents
  • Audio recordings
  • Braille materials
  • Electronic text files
  • In-person assistance

Contact our accessibility coordinator to request alternative formats at no additional cost.

Ongoing Efforts

We are committed to ongoing accessibility improvements, including:

  • Regular accessibility audits and testing
  • Staff training on accessibility best practices
  • User testing with people who use assistive technologies
  • Incorporating accessibility into our development process
  • Staying current with accessibility standards and guidelines

Legal Framework

This accessibility statement is aligned with:

  • Americans with Disabilities Act (ADA)
  • Section 508 of the Rehabilitation Act
  • WIOA Section 188 nondiscrimination requirements
  • Web Content Accessibility Guidelines (WCAG) 2.2
